Configuring the Hardware

PIC32MZ Embedded Connectivity with FPU (EF) Starter Kit

  • Press the switch SW1 to change the operating role.
  • Initially application will not operate in any role. Pressing SW1 for the first time will switch the application to Device mode. Press the switch again to switch the role between USB Host and Device.
  • Device mode operation
    • Connect the device to the Type A connector J4, which is located on the bottom side of the starter kit.
    • The application will emulate a USB HID mouse function. The cursor on the PC will rotate. Pressing SW1 will enable and disable the cursor movements.
  • In Host mode application
    • LED1 indicates USB Device Configuration Set Complete event (the USB device functionality has been activated by the USB Host).
    • Use the micro-A/B port J4 (which is located on the bottom side of the board) to connect the USB Device to the the USB Host PC.
