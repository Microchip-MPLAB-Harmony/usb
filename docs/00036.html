Running the Demonstration

This application demonstrates the capability of a PIC32 microcontroller and the MPLAB Harmony USB Host stack to work with two USB Controllers in an application. This application operates USB1 controller on the PIC32MK in USB Host mode and USB2 Controller in USB Device mode. 

 

USB1 controller on the PIC32 to operate as USB Host MSD and USB2 peripheral to appear like a serial (COM) port to the host. Do the following to run this demonstration:

  • Compile and program the target device. Refer to Building the Application for details.
  • Attach a USB Flash drive to the Host connector. Refer to Configuring the Hardware for details.
    • Refer to the Running the Demonstration section of the USB Host msd_basic demonstration for details on exercising the MSD Host features.
  • Connect USB Device connector to a USB Host. Refer to Configuring the Hardware for details.
    • Refer to the Running the Demonstration section of the USB Device cdc_com_port_single demonstration for details on exercising the CDC device features.
