Release Contents

This table lists the contents of the USB v3.6.0 release, including a brief description. 

 

USB Middleware:  

Module 
Description 
USB Device Stack 
USB Device Libraries with support for different USB device classes (CDC, Audio, HID, MSD, Printer and Vendor). 
USB Host stack 
USB Host Libraries with support for different client drivers (CDC, Audio, HID, and MSD) 

 

USB Drivers:

Module 
Description 
USBHS 
USB High Speed Device and Host mode driver for PIC32MZ EF/DA microcontroller families. 
USBFS 
USB Device and Host mode driver for PIC32MX/MK and PIC32MZ W1 microcontroller families. 
USBHSV1 
USB High Speed Device and Host mode driver for SAME70/V70/V71/S70 microcontroller families. 
USBFSV1 
USB Device and Host mode driver for SAMD2x/E5x/D5x/DA1/L21 microcontroller families.
USB Device mode driver for SAMD11/L22 microcontroller families. 
UDPHS 
USB High Speed Device Port driver for SAMA5D2 and SAM9X60 microprocessors. 
USBDP 
USB Device Port driver for SAMG55 microcontroller families. 
UHP-EHCI 
USB Host Port driver for EHCI (Enhanced Host Controller Interface) complaint USB peripheral in SAMA5D2 and SAM9X60 microprocessors. 
UHP-OHCI 
USB Host Port driver for OHCI (Open Host Controller Interface) complaint USB peripheral in SAMA5D2 and SAM9X60 microprocessors
and SAMG55 microcontrollers.
