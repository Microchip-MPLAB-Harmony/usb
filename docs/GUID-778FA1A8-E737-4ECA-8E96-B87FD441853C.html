1.1 Release Contents

This table lists the contents of the USB v3.13.0 release, including a brief description.

USB Middleware

Module Description
USB Device Stack The USB Device Libraries offer comprehensive support for various USB device classes, including CDC, HID, MSD, Audio, Printer, and Vendor.
USB Host stack USB Host Libraries offer support for various client drivers, including CDC, HID, MSD, Hub, and Audio.

USB Drivers

Module Description
USBHS USB High-Speed Device and Host mode driver for PIC32CZ CA80/CA90, PIC32CK, and PIC32MZ EF/DA microcontroller families.
USBFS USB Device and Host mode driver for PIC32MX/MK and PIC32MZ W1 microcontroller families.
USBHSV1 USB High-Speed Device and Host mode driver for SAME70/V70/V71/S70 microcontroller families.
USBFSV1 USB Device and Host mode driver for SAMD2x/E5x/D5x/DA1/L21, PIC32CM/CK/CX microcontroller families. USB Device mode driver for SAMD11/L22 microcontroller families.
UDPHS USB High-Speed Device Port driver for SAMA5, SAMA7, SAM9X60, and SAM9X7 microprocessor families.
USBDP USB Device Port driver for SAMG55 microcontroller family.
UHP-EHCI USB Host Port driver for EHCI (Enhanced Host Controller Interface) compliant USB peripheral in SAMA5, SAMA7, SAM9X60, and SAM9X7 microprocessor families.
UHP-OHCI USB Host Port driver for OHCI (Open Host Controller Interface) compliant USB peripheral in SAMA5, SAMA7, SAM9X60, and SAM9X7 microprocessor families, and SAMG55 microcontrollers.
